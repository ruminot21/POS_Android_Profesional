# NovaPOS

Sistema de punto de venta PWA para Android, tablet y PC.

## Acceso
- Administrador: `admin`
- Contraseña: `1234`
- Cajero: `cajero`
- Contraseña: `1234`

## Cómo abrir
1. Sube todos los archivos a GitHub Pages, Netlify o cualquier hosting.
2. Abre `index.html`.
3. En Android, abre la web con Chrome.
4. Presiona "Instalar aplicación" o usa el menú de Chrome > Instalar aplicación.

## Funciones
- Ventas y carrito
- Inventario y stock mínimo
- Productos y categorías
- Clientes
- Apertura, gastos y cierre de caja
- Reportes
- Usuarios administrador/cajero
- Copia de seguridad JSON
- Trabajo sin conexión
- Diseño adaptable

## Importante
Este proyecto usa almacenamiento local del navegador. Para uso comercial con varios dispositivos se recomienda conectar una base de datos y autenticación en servidor.


Incluye impresión de tickets y administración completa de usuarios.
